# Permeability-PAMPA
Algoritmos de prediccion de la permeabilidad en PAMPA con tecnicas de machine learning y XAI

Orden de trabajo
1. EDA con la base de datos con 780 DM y Fingerprints
2. weka_cmd1.py con los 50 seleccionados de EDA
3. Seleccion de atributos por consenso (11 Descriptores finales)
4. Arbol, RF, SVM, Regresion y k-NN
